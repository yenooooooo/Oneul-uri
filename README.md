## 📌 프로젝트 소개

**오늘우리**는 커플이 일상을 함께 기록하고 소통할 수 있는 웹 앱입니다.
데이트 기록, 공유 캘린더, 손편지, 저축 목표, 매일 커플 질문 등 연인 사이에 필요한 모든 기능을 하나의 앱에 담았습니다.

- **타겟:** iOS 아이폰 사용자 커플 (PWA → 홈 화면 추가로 네이티브 앱처럼 사용)
- **디자인 컨셉:** 비트윈 앱 스타일 — 따뜻하고 둥근 UI, 코랄 핑크 테마
- **배포:** Vercel (서버리스)

### 📝 데이트 기록

커플의 데이트를 사진, 장소, 감정과 함께 타임라인으로 기록합니다.

- **타임라인 뷰** — 월별 그룹핑으로 기록 목록 표시
- **사진 기록 카드** — 그라데이션 오버레이의 사진 카드 / 메모 스타일 카드
- **기록 작성** — 제목, 날짜, 장소, 감정(😍😊🤩🥰😂😴😢), 사진(최대 5장), 메모
- **기록 상세** — 사진 갤러리 + 메타데이터 + 댓글
- **검색** — 제목, 장소, 메모 내용으로 실시간 필터링
- **댓글** — 기록에 커플 간 댓글 작성
- **무한 스크롤** — 20개씩 페이지네이션
- **캐시 최적화** — Stale-while-revalidate 패턴으로 빠른 로딩

---

### 📅 공유 캘린더

커플이 함께 일정을 관리하는 인터랙티브 캘린더입니다.

- **월간 캘린더 그리드** — 이전/다음 달 네비게이션 + 오늘 버튼
- **날짜별 마커** — 일정, 데이트 플랜, 기록, 기념일 시각적 표시
- **스티커 꾸미기** — 64종 이상의 스티커로 날짜 데코레이션
- **선택 날짜 상세** — 해당 날짜의 플랜, 일정, 기록 한눈에 보기
- **일정 관리** — 일정 추가/수정/삭제 (날짜, 제목, 시간, 카테고리, 메모)
- **다가오는 기념일 칩** — 가로 스크롤 D-day 뱃지

---

### 📋 데이트 플래너

데이트 코스를 미리 계획하고 실행 후 기록으로 전환할 수 있습니다.

- **플랜 생성** — 제목 + 날짜로 새 플랜 생성
- **타임라인 뷰** — 시간순 정렬된 일정 항목 (제목, 시간, 장소, 메모, 비용)
- **체크리스트** — 각 항목별 완료 체크박스
- **플랜 완료** — 완료 시 데이트 기록으로 변환 가능
- **항목 추가/삭제** — 동적으로 코스 편집

---

### 🎂 기념일 관리

중요한 날을 놓치지 않도록 자동 & 수동으로 기념일을 관리합니다.

- **자동 생성** — 100일, 200일, 300일 ... 1000일, 1주년 ~ 5주년
- **생일 자동 등록** — 프로필에 생일 입력 시 자동 기념일 생성
- **커스텀 기념일** — 직접 추가 (제목, 날짜, 반복 여부)
- **D-day 뱃지** — D-DAY / D-N 형식으로 남은 일수 표시
- **다가오는 / 지난 기념일** — 섹션 분리 표시

### 💌 펜팔 (손편지)

디지털 손편지로 진심을 전하는 감성 소통 기능입니다.

- **편지지 선택** — 5종 편지지 (기본, 꽃, 별, 노트, 크라프트)
- **손글씨 폰트** — 나눔손글씨체로 자필 편지 느낌
- **사진 첨부** — 편지에 사진 1장 첨부 가능
- **답장 스레딩** — 원본 편지 인용 + 답장 연결
- **봉투 열기 연출** — 편지 읽기 모달 UI
- **읽음 표시** — 안 읽은 편지 뱃지 카운트
- **받은/보낸 편지함** — 탭으로 분류
- **무한 스크롤** — 페이지네이션 지원

### ❓ 매일 커플 질문

매일 새로운 질문에 블라인드로 답변하고, 서로의 답을 확인합니다.

- **매일 3개 질문** — KST 자정 기준 자동 갱신
- **3가지 답변 유형**
  - ✍️ 서술형 (자유 텍스트)
  - 🅰️🅱️ 양자택일 (A/B 선택)
  - 📊 점수형 (1~10 슬라이더)
- **블라인드 답변** — 둘 다 답변해야 공개 (잠금 아이콘 + 대기 표시)
- **일치 감지** — 같은 답변이면 "💕 같은 답변!" 표시
- **지난 질문 아코디언** — 날짜별 접기/펼치기 + 완료 상태 뱃지
- **늦은 답변 지원** — 지난 질문도 나중에 답변 가능, 제출 즉시 공개
- **50개 이상 질문 풀** — 솔직히, 우리 사이, 만약에, TMI 등 다양한 카테고리

### 💰 데이트 저축 목표

커플이 함께 목표 금액을 모아가는 공동 저축 기능입니다.

- **목표 설정** — 제목, 목표 금액, 목표 날짜
- **비행기 트래커** — 비행기 모양 시각적 진행률 바
- **페이스 분석** — 목표 대비 현재 속도 분석 (순조로움 / 빠름 / 느림)
- **입금 기록** — 금액, 메모, 입금자 표시
- **마일스톤 축하**
  - 🌱 10% — "첫 발걸음"
  - 🚀 25% — "1/4 달성"

## 🔔 푸시 알림 (iOS 16.4+)

PWA를 홈 화면에 추가하면 네이티브 앱처럼 푸시 알림을 받을 수 있습니다.

| 알림 종류 | 발송 시점 |
|-----------|-----------|
| **데일리 요약** | 매일 오전 9시 (KST) — D-day + 다가오는 기념일 + 오늘 일정 |
| **새 편지 도착** | 상대방이 편지를 보냈을 때 |
| **테스트 알림** | 설정 페이지에서 수동 발송 |

- **Vercel Cron Job** — 매일 09:00 KST 자동 트리거
- **Web Push + VAPID** — 별도 서비스 없이 무료 발송
- **만료 구독 정리** — 비활성 엔드포인트 자동 삭제

## 🗄️ 데이터베이스 구조

Supabase PostgreSQL 기반, 총 **21개 테이블** — 모든 테이블에 RLS(Row Level Security) 적용

```
couples                  커플 스페이스 (프로필, 닉네임, 이모지, 상태)
date_records             데이트 기록 (제목, 장소, 감정, 사진, 메모)
record_comments          기록 댓글
date_plans               데이트 플랜
date_plan_items          플랜 세부 항목
anniversaries            기념일 (자동/커스텀/생일)
calendar_events          캘린더 일정
```
